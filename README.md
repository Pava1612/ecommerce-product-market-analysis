# E-Commerce Product Market Analysis

## Project Overview

This project analyzes an e-commerce dataset to understand sales trends, product performance, customer behavior, and payment patterns. The goal is to transform raw transactional data into meaningful insights that can help businesses make better decisions.

The analysis was performed using Python for data analysis and Power BI for dashboard visualization.

---

## Dataset

Dataset used in this project:

Brazilian E-Commerce Public Dataset by Olist
Source: Kaggle

https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

The dataset contains information about:

* Orders
* Customers
* Products
* Payments
* Sellers
* Customer Reviews

These tables were merged to create a complete view of the e-commerce operations.

## Business Questions

The project focuses on answering the following business questions:

**Sales Performance**

* What is the total revenue generated?
* How do sales change monthly?

**Product Performance**

* Which product categories generate the most orders?
* Which categories generate the highest revenue?

**Customer Behavior**

* Which cities generate the most orders?
* What is the customer retention rate?

**Customer Satisfaction**

* Which categories have the highest review ratings?
* Which categories have low ratings but high sales?

**Payment Analysis**

* What are the most used payment methods?
* What is the average order value?

## Key Insights

* A small number of product categories generate the majority of revenue.
* Sales fluctuate monthly, indicating seasonal demand patterns.
* Urban cities contribute the highest number of orders.
* Most customer reviews are positive (4–5 stars).
* Some product categories have high sales but lower customer ratings.
* Credit cards are the most commonly used payment method.

## Business Recommendations
